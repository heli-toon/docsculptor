
DOCSCULPTOR - FEATURE IDEAS & ROADMAP
=====================================

🚀 HIGH-IMPACT FEATURES
-----------------------
□ Batch Processing
  - Multiple file uploads at once
  - Bulk PDF generation with progress tracking
  - ZIP export for multiple files
  - Queue management system

□ Template System
  - Pre-designed PDF templates (Academic, Business, Resume, Technical)
  - Custom CSS injection for advanced styling
  - Template marketplace/gallery
  - User-created template sharing

□ Export Format Expansion
  - DOCX export (Office documents)
  - RTF export (Rich Text Format)
  - Plain text export
  - EPUB export (eBooks)
  - LaTeX export

□ Real-time Collaboration
  - Share documents via links
  - Live preview sharing (like CodePen)
  - Real-time collaborative editing
  - Version control for documents

📚 CONTENT ENHANCEMENT
---------------------
□ Advanced Markdown Features
  - Math equations (KaTeX support) [DONE]
  - Mermaid diagrams and flowcharts
  - Custom containers/callouts/admonitions
  - Footnotes and endnotes support
  - Cross-references and internal links

□ Document Metadata
  - Document title, author, keywords
  - Creation date, modification tracking
  - Custom headers/footers with variables
  - Document properties and tags

□ Table of Contents (Implemented)
  - Auto-generated TOC from headings [DONE]
  - Clickable TOC with page numbers [DONE]
  - Customizable TOC depth and styling
  - TOC position options (before content, sidebar)

🎨 UI/UX IMPROVEMENTS
--------------------
□ Editor Integration
  - Built-in markdown editor with syntax highlighting
  - Split-pane view (editor + preview)
  - Live preview while typing
  - Code folding and line numbers

□ Themes & Customization
  - Multiple preview themes (GitHub, Academic, Minimal, Dark)
  - Dark/light mode for PDF output
  - Custom color schemes and branding
  - Font selection and preview [DONE]
  - Table "pop out" aesthetics [DONE]

□ Keyboard Shortcuts (Implemented)
  - Ctrl+E for export [DONE]
  - Ctrl+O for file open [DONE]
  - Ctrl+S for save draft [DONE]
  - Quick settings toggle (Ctrl+,) [DONE]
  - Full keyboard navigation [IMPROVED]

□ Drag & Drop Enhancements
  - Reorder settings sections
  - Drag files directly onto preview
  - Image drag & drop with auto-embedding
